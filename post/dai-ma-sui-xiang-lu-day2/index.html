###有序数组的平方
题目链接
给你一个按 非递减顺序 排序的整数数组 nums，返回 每个数字的平方 组成的新数组，要求也按 非递减顺序 排序。

示例 1：

输入：nums = [-4,-1,0,3,10]
输出：[0,1,9,16,100]
解释：平方后，数组变为 [16,1,0,9,100]
排序后，数组变为 [0,1,9,16,100]
示例 2：

输入：nums = [-7,-3,2,3,11]
输出：[4,9,9,49,121]

提示：

1 <= nums.length <= 104
-104 <= nums[i] <= 104
nums 已按 非递减顺序 排序

进阶：

请你设计时间复杂度为 O(n) 的算法解决本问题

O(nlog⁡n)O(n\log n)O(nlogn)暴力

class Solution {
public:
    vector<int> sortedSquares(vector<int>& nums) {
        for (int i = 0; i < nums.size(); i++) {  // change the condition from `i <= nums.size()` to `i < nums.size()`
            long long temp = (long long)nums[i] * nums[i];  // use long long to store the squared value
            nums[i] = temp;  // safely assign back to int since the problem guarantees the squared result fits in an int
        }
        sort(nums.begin(), nums.end());
        return nums;
    }
};

O(n)O(n)O(n)方法：双指针法，因为原本的已经按非递减的顺序排列，所以∃i使得a<i,nums[a]<0,b≤i,nums[b]≤0\exist i 使得a<i,nums[a]<0,b\le i,nums[b]\le 0∃i使得a<i,nums[a]<0,b≤i,nums[b]≤0.而且因为$a^2\le b^2 <->abs(a)\le abs(b) $所以最大的肯定是两边。有点类似归并排序的合并。

class Solution {
public:
    vector<int> sortedSquares(vector<int>& nums) {
        int l=0;
        int r=nums.size()-1;
        vector<int> ans(r+1);
        for(int i=r;i>=0;i--)
        {
            if(l>r)break;
            if(pow(nums[l],2)>=pow(nums[r],2))
            {
                ans[i]=pow(nums[l],2);
                l++;
            }
            else
            {
                ans[i]=pow(nums[r],2);
                r--;
            }
        }
        return ans;
    }
};

长度最小子数组

题目链接
给定一个含有 n 个正整数的数组和一个正整数 target 。

找出该数组中满足其总和大于等于 target 的长度最小的 连续子数组 [numsl, numsl+1, ..., numsr-1, numsr] ，并返回其长度。如果不存在符合条件的子数组，返回 0 。

1.思路1：sum(n,m)=sum(0,m)−sum(0,n)sum(n,m)=sum(0,m)-sum(0,n)sum(n,m)=sum(0,m)−sum(0,n)先初始化求和数组sum，在O(n2)O(n^2)O(n2)下遍历左边和右边

class Solution {
public:
    int minSubArrayLen(int target, vector<int>& nums) {
        vector<int> sum(nums.size()+1);
        int s=0;
        sum[0]=0;
        for(int i=0;i<=nums.size()-1;i++)
        {
            s+=nums[i];
            sum[i+1]=s;
        }
        int min=nums.size()+1;
        for(int l=0;l<=nums.size();l++)
        {
            for(int r=l+1;r<=nums.size();r++)
            {
                if(sum[r]-sum[l]>=target)
                {
                    if(r-l<min)
                    min=r-l;
                }
            }   
        }
        if(min>nums.size())min=0;
        return min;
    }
};

2. 滑动窗口
通过两个指针，左指针表示滑动窗口的起点，当区间满足时往前，并减去一个值
右指针表示滑动窗口的终止，遍历整个数组。

class Solution {
public:
    int minSubArrayLen(int target, vector<int>& nums) {
        int l=0;
        int length,sum=0;
        int ans=nums.size()+1;
        for(int r=0;r<nums.size();r++)
        {
            sum+=nums[r];
            while(sum>=target)
            {
                length=r-l+1;
                if(length<ans)ans=length;
                sum-=nums[l++];
            }
        }
        if(ans==nums.size()+1)
        ans=0;
        return ans;
    }
};

螺旋矩阵

题目链接
给你一个正整数 n ，生成一个包含 1 到 n2 所有元素，且元素按顺时针顺序螺旋排列的 n x n 正方形矩阵 matrix 。

一圈一圈循环。注意模拟的区间。

class Solution {
public:
    vector<vector<int>> generateMatrix(int n) {
        vector<vector<int>> ans(n,vector<int> (n,0));
        int startx=0;
        int starty=0;
        int loop=n/2;
        int mid=n/2;
        int offset=1;
        int count=1;
        int i,j;
        while(loop--)
        {
            
             for (j = starty; j < n - offset; j++) {
                ans[startx][j] = count++;
            }
            // 模拟填充右列从上到下(左闭右开)
            for (i = startx; i < n - offset; i++) {
                ans[i][j] = count++;
            }
            // 模拟填充下行从右到左(左闭右开)
            for (; j > starty; j--) {
                ans[i][j] = count++;
            }
            // 模拟填充左列从下到上(左闭右开)
            for (; i > startx; i--) {
                ans[i][j] = count++;
            }

            // 第二圈开始的时候，起始位置要各自加1， 例如：第一圈起始位置是(0, 0)，第二圈起始位置是(1, 1)
            startx++;
            starty++;

            // offset 控制每一圈里每一条边遍历的长度
            offset += 1;
        }
        if(n%2)
        {
            ans[mid][mid]=count;
        }
        return ans;
    }
};
